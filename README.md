# Claude Context Keeper

MCP-сервер для командной работы с Claude Code на одном сервере.
Сохраняет контекст между сессиями и разработчиками.

## Возможности

- Буферизованное логирование
- Чекпоинты с привязкой к Git
- Командный режим — видно кто что делал
- Детектор прерванных задач
- Атомарный fallback при сбое SQLite
- WAL mode с автоочисткой
- Мигратор старых проектов
- Блокировки чекпоинтов

## Установка

cd claude-context-keeper
./install.sh

## Инструменты MCP

migrate_project — миграция старого проекта
start_task — начать задачу
log — записать действие
log_decision — записать решение
complete_task — завершить и создать чекпоинт
restore_context — восстановить контекст
regenerate_claude_md — обновить claude.md
